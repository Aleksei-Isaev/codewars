# Is it a number

### Description
Given a string s, write a method (function) that will return true if it's a valid single integer or floating number or false if it's not.

Valid examples, should return `True`:

```python
isDigit("3")
isDigit("  3  ")
isDigit("-3.23")
```

should return `False`:

```python
isDigit("3-4")
isDigit("  3   5")
isDigit("3 5")
isDigit("zero")
```
