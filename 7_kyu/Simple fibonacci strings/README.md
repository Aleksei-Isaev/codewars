# Simple fibonacci strings

## Description

Given that:

```go
f0 = '0'
f1 = '01'
f2 = '010' = f1 + f0
f3 = '01001' = f2 + f1
```

You will be given a number and your task is to return the `nth` fibonacci string. For example:

```go
solve(2) = '010'
solve(3) = '01001'
```

More examples in test cases. Good luck!
