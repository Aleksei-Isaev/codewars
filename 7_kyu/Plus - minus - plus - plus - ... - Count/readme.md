# Plus - minus - plus - plus - ... - Count

## Description

Count how often sign changes in array.

### Result

number from `0` to ... . Empty array returns `0`

### Example

```python
const arr = [1, -3, -4, 0, 5]

| elem | count |
|------|-------|
|  1   |  0    |
| -3   |  1    |
| -4   |  1    |
|  0   |  2    |
|  5   |  2    |

return 2;
```
